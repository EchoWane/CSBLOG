### How to Contribute

Interested in co-authoring an article?
Contact me first to discuss potential topics and collaboration. Once we agree on a direction, we'll work together through the Github repo - you can fork, make your changes, and submit a pull request.

1. Fork the repo
2. Add your info in `src/data/authors/your-name.md`
3. Create your post in `src/data/blog/[language]/your-post-title.md`
4. Follow the frontmatter format from existing posts
5. Submit a PR with your post
